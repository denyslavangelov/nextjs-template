# UI / Design Rules

- Style with Tailwind only.
- Follow minimal, premium design (Apple-like).
- Use consistent spacing (no random values).
- Prefer whitespace over clutter.
- Use soft shadows and subtle borders.
- Avoid heavy colors unless intentional.
- Keep UI calm, clean, and readable.

Components:
- Buttons must look clickable.
- Inputs must have labels.
- Maintain visual hierarchy.

Typography:
- Limit font sizes.
- Use clear contrast.
- Avoid excessive bold usage.

Interaction:
- Add subtle hover states.
- Keep animations minimal and smooth.

Do not:
- overdesign
- clutter UI
- mix too many styles
